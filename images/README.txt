HOW TO ACTIVATE:
1. Drop your images in this folder
2. Open index.html in Notepad or VS Code
3. Find the <!-- Replace: ... --> comments
4. Remove the <!-- and --> around the <img> tags
5. Delete the <div class="slide-grid"></div> line (the placeholder grid)
6. Save and refresh browser

TIP: For logos, SVG format is best (tiny file size, scales perfectly).
     Get free brand SVGs from svgl.app or simpleicons.org
